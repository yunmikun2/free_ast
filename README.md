# FreeAst

Implements something similar to Free Monad. See examples directory for
more information.

## Usage

See [hexdocs](https://hexdocs.pm/free_ast) documentation for details.

## Installation

Add the following code to your `mix.exs`:

```elixir
def deps do
  [
    {:free_ast, "~> 0.3"}
  ]
end
```
